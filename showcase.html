Lines (RIFT)

// Simplified implementation
use websocket, db

let chat = websocket.Server()

chat.on("message", fn(msg, user) => {
  db.save(msg)
  chat.broadcast(msg, user.room)
})

chat.on("typing", fn(user) =>
  chat.emit("typing", user, user.room)
)

Community Contributions

Share your RIFT projects and templates with the community

Built something cool with RIFT? Share it with the community! 

# 1. Create a showcase entry
rift showcase init my-project

# 2. Add project details
#    - Description
#    - Code metrics
#    - Example code

# 3. Submit to showcase
rift showcase submit
What to Include:
  • Project name and description
  • Lines of code and reduction metrics
  • Key features and use cases
  • Example code snippets
  • Installation instructions
  • Links to source/documentation
Note: All metrics should be verifiable. Include comparison data where possible.
Contribution Guide

üìã Showcase Guidelines

Project Requirements:
  • Written in RIFT or uses RIFT significantly
  • Open source or publicly accessible
  • Includes accurate code metrics
  • Has documentation or README
  • Follows RIFT best practices
Metrics to Include:
  • Lines of Code: Total LOC in your implementation
  • Code Reduction: % reduction vs comparable implementation
  • Complexity: Big O notation where applicable
  • Performance: Benchmarks if relevant
Important: Do not include fake usage statistics, download counts, or star ratings. Focus on what your project enables, not adoption metrics.
