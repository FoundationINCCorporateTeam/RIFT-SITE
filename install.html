Once WSL is set up, open your WSL terminal and run:

Troubleshooting

Solutions to common installation issues

üîí Permission Denied

If you see "Permission denied" during installation: 

# Run with sudo (may require password)
curl -fsSL https://rift.astroyds.com/rift/install.sh | sudo sh

# Or install to user directory (no sudo needed)
curl -fsSL https://rift.astroyds.com/rift/install.sh | sh -s -- --prefix=$HOME/.local

üõ§Ô∏è Command Not Found

If rift command is not found after installation: 

# Add RIFT to your PATH (bash/zsh)
echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

# For fish shell
set -Ua fish_user_paths /usr/local/bin

# Then verify
rift --version

üåê Network Issues

If download fails or times out: 

# Try with explicit timeout and retry
curl --retry 3 --retry-delay 5 -fsSL \
  https://rift.astroyds.com/rift/install.sh | sh

# Or download and inspect first
curl -fsSL https://rift.astroyds.com/rift/install.sh > install.sh
cat install.sh  # review the script
sh install.sh

üèóÔ∏è Architecture Issues

If your architecture is not detected: 

# Check your system architecture
uname -m

# Supported: x86_64, arm64, aarch64
# If unsupported, please file an issue:
# https://github.com/FoundationINCCorporateTeam/RIFT-SITE/issues

Advanced Installation

Additional options for power users

üì¶ Custom Install Location

Install RIFT to a custom directory: 

# Install to custom prefix
curl -fsSL https://rift.astroyds.com/rift/install.sh | \
  sh -s -- --prefix=$HOME/.local

# Or any custom path
curl -fsSL https://rift.astroyds.com/rift/install.sh | \
  sh -s -- --prefix=/opt/rift

üîß Offline Installation

Install without internet during execution: 

# Download installer script
curl -fsSL https://rift.astroyds.com/rift/install.sh \
  -o rift-install.sh

# Review and run offline
chmod +x rift-install.sh
./rift-install.sh

üîÑ Version Management

Install specific versions or manage multiple installations: 

# Install specific version (future feature)
curl -fsSL https://rift.astroyds.com/rift/install.sh | \
  sh -s -- --version=1.0.0

# Check for updates
rift --check-update

# View installed version
rift --version

üè¢ Enterprise Deployment

Automated deployment for teams: 

# Silent installation (no prompts)
curl -fsSL https://rift.astroyds.com/rift/install.sh | \
  sh -s -- --silent

# With verification
curl -fsSL https://rift.astroyds.com/rift/install.sh | \
  sh && rift --version
